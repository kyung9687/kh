	자료를 정리하기 위해
	사용하는 태그


-	table 태그는
	행, 열을 작성하기 위한
	영역(판)을 지정하는 태그

  2. tr(table row) 태그

    -	테이블의 한 행을 나타내는 태그

  3. td(table Date) 태그

    -	테이블에 출력될 값(데이터)을
	작성하는 태그
-	각행의 열(column)역할
	== excel의 한 cell(한 칸)과 동일

  • th(Table) 태그

    		- 열의 제목을 나타내는 태그

		- td와 같은 레벨로 작성(td 대신 th 작성 가능)
		
		- 굵은 글씨 + 가운데 정렬

  • caption 태그

    		- 표 또는 이미지의 설명을 작성하는 태그

  • thead 태그

    		- 테이블 상단 영역(컬럼 제목 영역)

  • tbody 태그

    		- 테이블 중단 영역(값 작성 영역)

  • tfoot 태그

    		- 테이블 하단 영역(합계 작성 영역)

  • colspan(열 병합) 속성

    열로 나열된 방향(가로,옆) 병합

  • rowspan(행 병합) 속성

    행으로 나열된 방향(세로,위/아래) 병합

    • 테이블 연습1

    1행 1열 1행 2열 1행 3열
    2행 1열 2행 2열 2행 3열
    3행 1열 3행 2열 3행 3열
